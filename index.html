What is MamaPut App?

MamaPut App is a food delivery application designed to connect users with local food vendors, offering a wide range of traditional dishes. The app aims to support local businesses while providing users with a convenient way to enjoy their favorite meals without leaving their homes. It has both a front end and a back end api developed with python flask.

Front End Key Features

  • User-Friendly Interface: Built with React.js, the app boasts a clean and intuitive interface, making it easy for users to navigate and place orders.
  • Local Cuisine Focus: Unlike many other food delivery apps, MamaPut App emphasizes local dishes, giving users access to authentic and traditional meals.
  • Efficient Delivery System: The app ensures timely delivery, maintaining the freshness and quality of the food.
  • Responsive Design: The app is designed to work seamlessly across various devices, ensuring a smooth experience whether you're on a smartphone, tablet, or desktop.

Back End Technical Overview

MamaPut Api leverages the power of python to create a dynamic and responsive user interface. Here are some of the technical highlights:

  • Python 3.11: The core language used for it implemetation. I used class routes which present so much ease in it usuage
  • Flask-SqlAlchemy: Flask-SQLAlchemy is a PyPI package that simplifies using SQLAlchemy with Flask by providing useful defaults and extra helpers
  • Pipenv: is a Python virtualenv management tool that supports a multitude of systems and nicely bridges the gaps between pip, python (using system python, pyenv or asdf) and virtualenv.
  • Marshmallow Schemas: is used for Validate input data, Deserialize input data to app-level objects and Serialize app-level objects to primitive Python types. The serialized objects can then be rendered to standard formats such as JSON for use in an HTTP API.
